# Power of AST - PHP Edition

## Requirements

| Name     | Version            |
|----------|--------------------|
| PHP      | >= 7.4.0 && < 8.2  |
| Composer | 2.x.x              |

## Setup

```sh
composer install
composer dump-autoload
```

## Running Linter

```sh
vendor/bin/php-cs-fixer fix
```
